\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{md_README_autotoc_md1}
{\ttfamily inverter.\+cxx} defines a small C++ program and a {\ttfamily \doxylink{classPPM}{PPM}} image container for reading \doxylink{classPPM}{PPM} images ({\ttfamily P3} or {\ttfamily P6}) from standard input and printing the parsed header and pixel values to standard output. The code enforces bounds on image dimensions and color depth.\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item Supports \doxylink{classPPM}{PPM} magic numbers {\ttfamily P3} (ASCII) and {\ttfamily P6} (binary)
\item Enforces limits on width, height, and maximum color value
\item Outputs image metadata and pixel values
