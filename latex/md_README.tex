{\ttfamily \doxylink{inverter_8cxx}{inverter.\+cxx}} defines a small C++ program and a {\ttfamily \doxylink{classPPM}{PPM}} image container for reading \doxylink{classPPM}{PPM} images ({\ttfamily P3} or {\ttfamily P6}) from standard input and printing the parsed header and pixel values to standard output. The code enforces bounds on image dimensions and color depth.\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item Supports \doxylink{classPPM}{PPM} magic numbers {\ttfamily P3} (ASCII) and {\ttfamily P6} (binary)
\item Enforces limits on width, height, and maximum color value
\item Stream extraction operator ({\ttfamily operator\texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>}}) for loading \doxylink{classPPM}{PPM} data
\item Outputs image metadata and pixel values line by line
\end{DoxyItemize}\hypertarget{md_README_autotoc_md3}{}\doxysection{\texorpdfstring{Build}{Build}}\label{md_README_autotoc_md3}
